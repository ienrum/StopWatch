# Stop Watch 웹 애플리케이션

![Stop Watch 스크린샷](./screen.png){: width="50%" height="50%"}

## 소개

Stop Watch는 간단한 스톱워치 시간 관리 웹 애플리케이션입니다. 사용자는 시간을 추가, 삭제 할 수 있으며, 완료한 작업을 저장할 수 있습니다. 또한 통계그래프를 볼수 있습니다.

## 주요 기능

- 시간 추가, 삭제
- 통계 그래프
- 스톱 워치

## 사용된 기술

- Front-end: React, css modules, javascript
- Deployment: netlify
